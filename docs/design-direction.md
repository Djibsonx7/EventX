# EventX – Design Direction

## Design ambition
EventX should feel premium, modern, and event-native.

It must create an immediate **wow effect** for organizers, sponsors, and audiences, while staying practical for real-world event execution.

## Core visual principles

### 1. Event-tech premium feel
- dark, immersive surfaces
- luminous accent gradients
- strong contrast for stage readability
- modern typography
- animated but controlled motion

### 2. Mobile-first and stage-ready
- readable on smartphones
- visually strong on projector screens
- CTA hierarchy clear from a distance

### 3. Sponsor-native visual system
- sponsor placements must feel integrated, not forced
- media carousels, transition cards, reward highlights
- premium sponsor visibility treatment

### 4. Modular experience language
Every game mode should feel part of one ecosystem, not a random collection of mini-games.

## Design references (conceptual)
- premium SaaS dashboards
- event-tech products
- modern sports/entertainment overlays
- live-show scoreboards

## Initial color direction
- deep midnight / dark indigo base
- electric violet / blue gradients
- warm gold accent for rewards and premium cues
- emerald / neon mint for success states

## Typography direction
- headline: bold, high-impact, geometric
- UI text: clean, legible, modern sans-serif
- numeric displays: strong, scoreboard-like presence

## Motion direction
- subtle entrance transitions
- strong hero emphasis for live screens
- responsive feedback for taps, countdowns, and rewards
- no excessive motion clutter

## Core surfaces to design first
1. Landing / homepage
2. Organizer dashboard shell
3. Session creation flow
4. Live waiting room
5. Quiz battle session screen
6. Poll screen
7. Word cloud screen
8. Spin & Win screen

## Design promise
EventX should look like a product that belongs on a big stage, not just inside a small app.

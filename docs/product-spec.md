# EventX – Product Specification

## Overview
EventX is a live audience engagement platform designed for events, conferences, brand activations, educational environments, and collective experiences.

It transforms passive audiences into active participants using simple, mobile-first interactions with strong visual feedback on big screens.

## Product summary
EventX is not just a quiz app.
It is a B2B live engagement platform that helps organizations create interactive moments for audiences in real time.

Its natural use cases include:
- events and ceremonies
- conferences and seminars
- brand activations and stands
- campuses and educational programs
- training sessions and workshops
- institutional and community experiences

## Target customers

### Primary customer profile
EventX is sold to an operating organization.
That organization can be:
- an event agency
- a company
- a school or university
- an institution or NGO
- a brand activation team

### Early go-to-market priority
Recommended early segments:
- event agencies
- companies running internal or external events
- schools / campuses / training organizations
- brand activation and field marketing teams

## Roles in the platform

### 1. Platform owner
The EventX owner manages the platform globally.

Owner responsibilities may include:
- client management
- licensing and billing
- feature access and plans
- global analytics
- platform supervision

### 2. Operator / client organization
This is the organization that uses EventX to create and run live experiences.

This role can be played by:
- an agency such as an event operator
- a company using EventX directly
- a school or institution using EventX directly

This operator sees:
- dashboard / organizer studio
- session creation
- game format selection
- sponsor configuration
- live session launch tools
- event results and analytics

### 3. Participant / player
This is the audience member who joins a live session.

This role sees:
- join with code / QR
- live game or interaction
- score / result / ranking
- very light and mobile-first UX

## Product principle for MVP
For simplicity, the MVP treats one organization as one operating account.

That means:
- one operator organization creates and runs sessions
- participants join those sessions
- the platform owner remains above all organizations

Advanced agency hierarchies such as sub-clients, nested workspaces, or white-label partner structures are postponed to a later phase.

## Business model direction
EventX should not be treated as a one-time software purchase.
The recommended model is a renewable license / subscription.

Possible pricing structures later:
- monthly or yearly subscription
- subscription + usage limits
- event packs / usage credits
- premium modules for sponsor features or analytics

## Partner channel direction
EventX can be sold both:
- directly to organizations
- through agencies as operating partners

For the MVP, EventX remains open to any operating organization.
Potential partner protections, white-label options, and multi-client agency structures are future evolutions, not MVP blockers.

## Core flows

### 1. Organizer creates a session
- selects format (quiz, poll, word cloud, spin)
- configures content
- adds sponsor elements
- generates join code / QR code

### 2. Audience joins
- mobile phone
- join code or QR scan
- no friction / minimal onboarding

### 3. Live interaction
- game / poll / input
- real-time updates
- leaderboard / visualization

### 4. Results & closure
- winner / summary
- sponsor visibility
- potential data export later

## Game formats (MVP)

### Quiz Battle
- timed questions
- scoring system
- leaderboard

### Live Poll
- multiple choice
- real-time visualization

### Word Cloud
- text input
- visual aggregation

### Spin & Win
- reward distribution
- sponsor integration

## Core value proposition
EventX helps organizations:
- engage audiences in real time
- energize live events and group experiences
- create sponsor-friendly moments
- differentiate their event experience
- measure participation and interaction quality

## What EventX brings concretely

### For an event agency
- premium live interactive layer to sell in client events
- differentiation versus competing agencies
- reusable experience formats across events
- sponsor integration opportunities

### For a company
- stronger seminar and conference engagement
- dynamic training or internal event formats
- measurable participation

### For a school / institution
- more interactive educational or community experiences
- campus competitions and engagement moments
- easy participation from mobile devices

### For a brand activation team
- stronger stand traffic and participation
- memorable sponsor moments
- reward-based interactions

## Sponsor system
Sponsor support is important, but it is not the only pillar.
The core of EventX is live audience engagement.
Sponsoring increases business value and differentiation.

Sponsor features include:
- logo placement
- media carousel (image/video)
- branded transitions
- sponsored rounds
- sponsored rewards

## ROI and analytics direction
For B2B adoption, EventX should prove value, not just create animation.

Important future metrics may include:
- participant count
- participation rate
- completion rate
- engagement time
- best-performing formats
- sponsor visibility metrics
- response volume and interaction density

## UX principles
EventX requires different UX layers by role.

### Participant UX
- mobile-first
- ultra simple
- low friction
- fast onboarding
- clear live feedback

### Organizer UX
- premium and reassuring
- simple to configure
- easy to launch under event pressure
- results visible and actionable

### Owner UX
- supervisory and analytical
- focused on platform-wide visibility and business control

## Non-functional requirements
- mobile-first
- low latency
- scalable sessions
- clean UI for projection screens
- resilient session lifecycle

## Future expansions
- multi-room events
- analytics dashboard
- sponsor reporting
- integrations (CRM, ticketing, etc.)
- partner / agency multi-client structures
- white-label options
- AI-assisted content generation and recommendations

## Success criteria
- fast onboarding
- high participation rate
- strong visual impact
- sponsor satisfaction
- repeat usage by organizations

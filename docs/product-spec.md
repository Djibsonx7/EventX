# EventX – Product Specification

## Overview
EventX is a live audience engagement platform designed for events, conferences, brand activations, and educational environments.

It transforms passive audiences into active participants using simple, mobile-first interactions with strong visual feedback on big screens.

## Target users

### Primary
- Event organizers
- Event agencies
- Corporate communication teams
- Schools and universities

### Secondary
- Sponsors and brands
- Hosts / MCs

## Core flows

### 1. Organizer creates a session
- selects format (quiz, poll, word cloud, spin)
- configures content
- adds sponsor elements
- generates join code / QR code

### 2. Audience joins
- mobile phone
- join code or QR scan
- no friction / minimal onboarding

### 3. Live interaction
- game / poll / input
- real-time updates
- leaderboard / visualization

### 4. Results & closure
- winner / summary
- sponsor visibility
- potential data export later

## Game formats (MVP)

### Quiz Battle
- timed questions
- scoring system
- leaderboard

### Live Poll
- multiple choice
- real-time visualization

### Word Cloud
- text input
- visual aggregation

### Spin & Win
- reward distribution
- sponsor integration

## Sponsor system

- logo placement
- media carousel (image/video)
- branded transitions
- sponsored rounds
- sponsored rewards

## Non-functional requirements

- mobile-first
- low latency
- scalable sessions
- clean UI for projection screens
- resilient session lifecycle

## Future expansions

- multi-room events
- analytics dashboard
- sponsor reporting
- integrations (CRM, ticketing, etc.)

## Success criteria

- fast onboarding
- high participation rate
- strong visual impact
- sponsor satisfaction

# EventX

EventX is a premium live engagement platform for events, brands, conferences, schools, and activations.

## Vision

EventX helps organizers turn passive audiences into active participants through mobile-first live experiences designed for the local market:
- Quiz Battle Live
- Live Polls
- Word Cloud
- Spin & Win

## Why EventX

Unlike a single quiz product, EventX is designed as an event engagement suite:
- sponsor-native from day one
- mobile-first gameplay
- premium live-event visual identity
- adaptable to conferences, activations, campuses, ceremonies, and B2B events
- robust enough to evolve into a modular event-tech platform

## Product pillars

### 1. Engage the audience
Fast, simple, visible interactions that work on stage and on mobile.

### 2. Elevate sponsors
Native sponsor placements across lobbies, transitions, lots, and game rounds.

### 3. Fit the local market
Simple onboarding, low-friction participation, adaptable prize logic, and event-ready flows.

### 4. Scale by format
A shared session engine that can power multiple game types instead of one isolated experience.

## MVP modules

### Quiz Battle Live
Head-to-head or multiplayer timed quiz battles.

### Live Polls
Fast audience voting for conferences, seminars, and corporate sessions.

### Word Cloud
A lightweight live participation format for icebreakers, feedback, and sentiment capture.

### Spin & Win
A sponsor-friendly reward mechanic for stands, activations, and ceremonies.

## Core sponsor features
- sponsor logo and brand identity in session UI
- sponsor media carousel (images/video)
- sponsor-branded transitions and waiting screens
- sponsored rounds and sponsored rewards
- sponsor visibility reporting in later phases

## Product experience goals
- wow effect on first screen
- modern event-tech design
- strong readability on projector and mobile
- very low participation friction
- reusable engine for future game formats

## Proposed architecture direction

### Frontend
- React
- Tailwind CSS
- Framer Motion
- mobile-first responsive UX

### Backend
- Node.js / Express
- modular game/session engine
- WebSocket realtime layer
- sponsor and analytics-ready data model

### Data
- MongoDB or PostgreSQL depending on final domain model direction

## Suggested roadmap

### Phase 1
Product foundation, design system, app shell, and session model.

### Phase 2
Quiz Battle Live + sponsor-ready waiting room + live session flow.

### Phase 3
Live Polls + Word Cloud.

### Phase 4
Spin & Win + organizer dashboard improvements.

### Phase 5
Analytics, reporting, advanced sponsor visibility, and deployment hardening.

## Positioning statement

EventX is a premium audience engagement suite built for live events, designed to help organizers energize participants and help sponsors stand out.
